# Counter-App

A simple and interactive Counter Application built using **HTML**, **CSS**, and **JavaScript**.
This project demonstrates basic front-end development concepts such as DOM manipulation, event handling, and responsive UI design.

---

## 🚀 Features

* Increment Counter
* Decrement Counter
* Reset Counter
* Responsive Design
* Simple and Clean User Interface
* Beginner-Friendly Project

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
